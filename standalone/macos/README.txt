Eternal Hearts: A Ravencroft Romance - macOS Edition
===================================================

Welcome to the standalone version of Eternal Hearts: A Ravencroft Romance!

REQUIREMENTS:
- macOS 10.12 Sierra or later
- Node.js (will be automatically detected or prompt for installation)

HOW TO PLAY:
1. Double-click "Start Game.command" to launch the game
2. If prompted about security, go to System Preferences > Security & Privacy and click "Open Anyway"
3. The game will start a local server and open in your default web browser
4. If the browser doesn't open automatically, navigate to: http://localhost:3000
5. Enjoy your supernatural romance adventure!

TROUBLESHOOTING:
- If Node.js is not installed, download it from: https://nodejs.org
- If you get a security warning, right-click the file and select "Open"
- Make sure no other application is using port 3000
- Check that Terminal has permission to run scripts

TO STOP THE GAME:
- Close the browser tab
- Press Ctrl+C in the Terminal window
- Or simply close the Terminal window

GAME FEATURES:
- Complete vampire romance visual novel
- 1,565+ interconnected story scenes
- 9 datable supernatural characters
- Character creation with traits, flaws, and boons
- Save/load system with multiple slots
- Trust and affection relationship mechanics

Enjoy your journey into the world of Ravencroft Manor!
