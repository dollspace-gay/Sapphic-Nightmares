Eternal Hearts: A Ravencroft Romance - Universal Edition
========================================================

Welcome to the standalone version of Eternal Hearts: A Ravencroft Romance!

REQUIREMENTS:
- Any operating system (Windows, macOS, Linux)
- Node.js (will be automatically detected or prompt for installation)

HOW TO PLAY:

Windows:
- Double-click "Start Game.bat" if available
- Or open Command Prompt, navigate to this folder, and run: node game-launcher.js

macOS/Linux:
- Double-click "Start Game.sh" if your system supports it
- Or open Terminal, navigate to this folder, and run: ./Start\ Game.sh
- Or run directly: node game-launcher.js

The game will start a local server and open in your default web browser.
If the browser doesn't open automatically, navigate to: http://localhost:3000

TROUBLESHOOTING:
- If Node.js is not installed, download it from: https://nodejs.org
- Make sure no other application is using port 3000
- Check system security settings if scripts are blocked
- Run directly with: node game-launcher.js

TO STOP THE GAME:
- Close the browser tab
- Press Ctrl+C in the terminal/command window
- Or simply close the terminal/command window

GAME FEATURES:
- Complete vampire romance visual novel
- 1,565+ interconnected story scenes
- 9 datable supernatural characters
- Character creation with traits, flaws, and boons
- Save/load system with multiple slots
- Trust and affection relationship mechanics

Enjoy your journey into the world of Ravencroft Manor!
